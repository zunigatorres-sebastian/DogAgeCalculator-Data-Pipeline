# DogAgeCalculator-Data-Pipeline

An end-to-end BI solution that transforms raw clinical veterinary data into actionable insights. This project implements a full data pipeline, from SQL-based normalization to advanced interactive modeling in Power BI

![dashboard_main](assets/dashboard_main.png)

## Data Pipeline
C# Application → TXT Data Export → MySQL Database → SQL Data Cleaning → Power BI Dashboard

### Business Intelligence & KPIs
The interactive dashboard provides a 360° view of canine health metrics, fully responsive to breed-specific filtering:

- **Clinical Metrics:** Average Dog Age, Average Weight, and Average Converted (Human) Age
- **Risk Analysis:** Percentage of Older Dogs (Senior population tracking)
- **Statistical Correlation:** Weight Variance and Age-Weight distribution

### Visualizations
- **Breed Demographic Analysis:** Dynamic bar chart for population count per breed
- **Biometric Distribution:** Scatter plot analyzing the correlation between age and weight to identify developmental outliers

## Key Features
- **Data Engineering:** Implementation of SQL scripts for data cleaning, normalization, and outlier management in weight records
- **Advanced DAX Modeling:** Custom measures for dynamic conditional formatting and real-time calculation of clinical variances
- **Interactive UX:** Multi-selection slicers and cross-filtering capabilities for granular breed analysis
- **ETL Process:** Structured workflow for migrating raw .TXT exports into a relational MySQL schema

## Technologies
- **Database Engine:** MySQL (Data persistence and relational schema)
- **Languages:** SQL (Data Definition & Manipulation), DAX (Analytical modeling)
- **Visualization:** Power BI Desktop
- **Connectivity:** [**mysql-connector-net-9.7.0:**](https://dev.mysql.com/downloads/connector/net/) 

## How to Run the Project

*Note on Data Source: This pipeline is designed to process data samples generated by the DogAgeCalculator WinForms application, demonstrating the integration between custom software and BI tools*

1. Database Setup: Run the **dog_Query.sql** script in MySQL Workbench to start the **dogdb** database
2. Data Import: Move the **dog_sample_.txt** file to the MySQL secure folder. Path is "C:\ProgramData\MySQL\MySQL Server 8.0\Uploads\dog_sample_.txt"
3. Connector Installation: Install the MySQL Connector/Net to allow Power BI to communicate with the local database
4. Open Dashboard: Open the **DataAnalysis_DogDB.pbix** file. If necessary, update the data source credentials in "Transform Data" > "Data Source Settings"
